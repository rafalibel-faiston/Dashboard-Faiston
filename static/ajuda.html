Aba Carimbos

O que são carimbos?

Modelos de texto prontos para copiar e colar em outros sistemas — chamados, emails, SAP, etc. Cada equipe tem os próprios carimbos.

Clique em Copiar no card para copiar o texto para a área de transferência
Carimbos são agrupados por categoria com cores diferentes
Visíveis para toda a equipe do seu time

Criar / Editar Carimbos

Clique em "+ Novo Carimbo" e preencha título, categoria e conteúdo
O ícone de lápis aparece nos carimbos que você criou — clique para editar
Dentro do modal de edição, clique em Excluir para remover
Nomes de carimbos são únicos por time

Clique em Nova nota para criar e escolher um título
Salva automaticamente enquanto você digita

O que é o perfil Demo?

O perfil Demo é uma conta de teste que permite visualizar duas visões do sistema sem ser um usuário real de produção. Ideal para apresentações e treinamentos.

Visão Funcionário

É a visão padrão ao entrar. Mostra apenas as tarefas da conta demo — não vê tarefas de outros usuários. Acesso às abas Kanban, Agenda e Carimbos do time.

Visão Gestor

Clique em "Ver Visão Gestor" no banner roxo do topo. Mostra o Dashboard completo com todas as tarefas do time, filtros e gráficos.

Capacidades do perfil Demo

Criar, editar e concluir tarefas
Criar e gerenciar usuários do time
Criar carimbos para o time
Ver Dashboard, clientes e financeiro

Perfil Admin

O admin tem acesso a tudo: dashboard, clientes, financeiro + gerenciamento completo de usuários e ferramentas de manutenção. Acesse o painel exclusivo em /admin pelo link no menu lateral.

Gerenciar Usuários

Criar Usuário

  1. 1Clique em "Novo Usuário"
  2. 2Preencha nome, usuário, senha temporária, perfil, time e email
  3. 3Salve — o sistema envia automaticamente um email com as credenciais
  4. 4No primeiro login, o usuário cria sua senha pessoal

Perfis disponíveis

Adminacesso total + painel admin
Gestordashboard + clientes + financeiro + time
Funcionáriosó as próprias tarefas
Democonta de teste — visão funcionário + gestor

Editar / Remover Usuários

Ícone de lápis → editar nome, perfil, time, email ou redefinir senha
Ícone de envelope → reenviar email de acesso
Ícone de lixeira → remove o usuário (histórico preservado)
Ao redefinir senha, usuário é obrigado a trocar no próximo login
O usuário admin não pode ser removido.

Ferramentas de Manutenção

Corrigir Horas Corrompidas

O botão Corrigir Horas varre todas as tarefas e lista as que têm tempo absurdo (mais de 30 dias contínuos). Para cada uma, você insere as horas e minutos corretos e salva.

Times dos Carimbos

O botão Times dos Carimbos lista todos os carimbos com o time atual. Use para corrigir carimbos antigos que foram cadastrados antes da separação por equipe.

Limpar Todas as Tarefas

Ação irreversível

O botão Limpar Testes remove todas as tarefas do sistema. Use apenas para limpar dados de teste antes do uso real.

Carimbos — Admin

O admin vê o ícone de lápis em todos os carimbos, independente de quem criou. Pode editar e excluir qualquer carimbo de qualquer time.

Dicas Gerais

Tema Escuro

Clique no ícone 🌙 / ☀️ no canto superior direito. A preferência fica salva no navegador e se aplica a todas as telas.

Sessão Segura

Sua sessão dura 24 horas e fica salva mesmo que o sistema reinicie. Para sair, clique em "Sair" no canto superior direito.

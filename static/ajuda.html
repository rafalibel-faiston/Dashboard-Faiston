Primeiro Acesso

Na primeira vez que você logar com a senha temporária, o sistema vai pedir para criar uma senha pessoal. Mínimo 6 caracteres. Depois disso, é só usar normalmente.

Aba Kanban — Suas Tarefas

Criar uma tarefa

  1. 1 Clique em "Nova Tarefa"
  2. 2 Selecione o Projeto → cliente preenche sozinho
  3. 3 Defina Prazo e/ou Agendamento
  4. 4 Clique em Salvar

Controle de Tempo

  • Clique no ▶ play para iniciar o timer — status muda para Em Andamento
  • Clique em ■ stop para pausar e salvar o tempo
  • Para registrar horas manuais, abra a tarefa e preencha o campo Horas Manuais

Aba Agenda — Estilo Outlook

Como funciona

  • Pontos roxos no calendário = tarefas agendadas naquele dia
  • Pontos rosas = tarefas com prazo vencendo
  • Use ‹ › para navegar entre meses
  • Clique em qualquer dia para ver as tarefas daquele dia
  • Seção Atrasadas sempre aparece no topo

Diferença: Prazo vs. Agendamento

Prazo (até quando)

Data limite para entregar. Aparece como badge colorido na card.

Agendar para

Dia que você vai trabalhar nisso. Aparece na Agenda naquele dia.

Clientes & Projetos

Cadastrar Cliente

Clique em "Novo Cliente", preencha nome e setor. Cliente aparece no dashboard e nos filtros.

Projetos por Cliente

Na aba Clientes, clique em "Projetos" no card do cliente para ver, adicionar e editar projetos com orçamento.

Importar de Planilha

Botão "Importar Projetos" → cole link OneDrive → mapeie colunas → confirme. Só importa projetos novos, não duplica.

Perfil Admin

O admin tem acesso a tudo: dashboard, clientes, financeiro + gerenciamento de usuários no /admin. Acesse pelo link "Admin" no menu lateral.

Gerenciar Usuários

Criar Usuário

  1. 1 Clique em "Novo Usuário"
  2. 2 Preencha nome, usuário, senha temporária e perfil
  3. 3 Salve — o sistema vai pedir troca de senha no primeiro login
Perfis disponíveis:
Admin — acesso total + usuários
Gestor — dashboard + clientes + financeiro
Funcionário — só as próprias tarefas

Editar / Remover Usuários

  • Ícone de lápis → editar nome, perfil, status, ou redefinir senha
  • Ícone de lixeira → desativa o usuário (não apaga o histórico)
  • Ao redefinir senha → usuário é obrigado a trocar na próxima entrada
⚠️ O usuário admin não pode ser desativado.

Limpar Tarefas

Na aba "Tarefas do Time", o botão 🗑 Limpar Todas remove todas as tarefas do sistema de uma vez. Pede confirmação antes. Ação irreversível.

Dicas Gerais

Tema Escuro

Clique no ícone 🌙/☀️ no canto superior direito. A preferência fica salva no seu navegador.

Atualizar Dados

Use o botão 🔄 no topo para recarregar os dados sem precisar atualizar a página inteira.

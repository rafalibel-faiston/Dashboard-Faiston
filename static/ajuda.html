Primeiro Acesso

Na primeira vez que você logar com a senha temporária, o sistema vai pedir para criar uma senha pessoal. Mínimo 6 caracteres. Depois disso, é só usar normalmente.

Aba Kanban — Suas Tarefas

Criar uma tarefa

  1. 1 Clique em "Nova Tarefa"
  2. 2 Selecione o Projeto — o cliente preenche automaticamente
  3. 3 Defina o Prazo e/ou o Agendamento
  4. 4 Clique em Salvar

Controle de Tempo

Clique no ▶ play para iniciar — status muda para Em Andamento

Clique em ■ stop para pausar e salvar o tempo registrado

Para horas manuais, abra a tarefa e preencha o campo Horas Manuais

Clientes & Projetos

Cadastrar Cliente

Clique em "Novo Cliente", preencha nome e setor. O cliente aparece no dashboard e nos filtros automaticamente.

Projetos por Cliente

Na aba Clientes, clique em "Projetos" no card do cliente para ver, adicionar e editar projetos com orçamento.

Importar de Planilha

Botão "Importar Projetos" → cole o link OneDrive → mapeie as colunas → confirme. Não duplica projetos já cadastrados.

Perfil Admin

O admin tem acesso a tudo: dashboard, clientes, financeiro + gerenciamento de usuários em /admin. Acesse pelo link "Admin" no menu lateral.

Gerenciar Usuários

Criar Usuário

  1. 1Clique em "Novo Usuário"
  2. 2Preencha nome, usuário, senha temporária e perfil
  3. 3Salve — o sistema pedirá troca de senha no primeiro login

Perfis disponíveis

Adminacesso total + usuários
Gestordashboard + clientes + financeiro
Funcionáriosó as próprias tarefas

Editar / Remover Usuários

Ícone de lápis → editar nome, perfil ou redefinir senha
Ícone de lixeira → desativa o usuário (histórico preservado)
Ao redefinir senha, usuário é obrigado a trocar no próximo login
O usuário admin não pode ser desativado.

Limpar Todas as Tarefas

Ação irreversível

Na aba "Tarefas do Time", o botão 🗑 Limpar Todas remove todas as tarefas do sistema de uma vez. Uma confirmação é exibida antes. Use com cuidado.

Dicas Gerais

Tema Escuro

Clique no ícone 🌙/☀️ no canto superior direito. A preferência fica salva no seu navegador.

Atualizar Dados

Use o botão 🔄 no topo para recarregar os dados sem precisar atualizar a página inteira.
